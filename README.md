# Revised Aurelia Atlas vs3
Revised Scripts necessary to reproduce the analyses in Link et al. BioRxiv [https://doi.org/10.1101/2023.08.24.554571v4]
 - published in PLOS Biology (2026)

1) Linketal.Ac.generate.data_REVISED.R : R script for importing mapped count matrices from Cellranger 'out' directory after mapping and generating integrated Seurat R object for analysis

2) Linketal.Ac.analyse.alldata_REVISED.R : R script for analyses of life cycle stages

3) Linketal.Ac.analyse.subsets_REVISED.R : R script for analyses of clustering 

4) Linketal.Ac.generate.figures_REVISED.R : R script for generating the figures

5) Linketal.Ac.Nv.OMA_REVISED.R : R script for running comparative analysis with *Nematostella vectensis*

6) Supplement.rmd : R markdown document for the supplementary figures
