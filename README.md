# AureliaAtlas

Scripts and files necessary to reproduce the analyses in Link et al. BioRxiv*

1) GenerateAaAllData.R : R script for importing mapped count matrices from Cellranger 'out' directory after mapping and generating integrated Seurat R object for analysis

2) Aurelia51k.Atlas.LifeHistoryStages_clean.RMD : R markdown document for analyses of life cycle stages

3) Aurelia51k.Atlas.Alldata_clean.RMD : R markdown document for clustering of the full dataset

4) Aurelia51k.Atlas.CellComplement_clean.RMD : R markdown document for iterative analysis of each principle cluster or partition in the dataset
