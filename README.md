# AureliaAtlas
Scripts and files necessary to reproduce the analyses in Link et al. BioRxiv*

GenerateAaAllData.R : R script for importing mapped count matrices from Cellranger 'out' directory after mapping and generating integrated Seurat R object for analysis
Aurelia51k.Atlas.LifeHistoryStages_clean.RMD : R markdown document for analyses of life cycle stages
Aurelia51k.Atlas.Alldata_clean.RMD : R markdown document for clustering of the full dataset
Aurelia51k.Atlas.CellComplement_clean.RMD : R markdown document for iterative analysis of each principle cluster or partition in the dataset
